Условные операторы

В C# существует несколько типов условных выражений, которые позволяют управлять логикой выполнения программы в зависимости от определенных условий:


Условный оператор if-else:

- Основная структура - if (условие) { действие1 } else { действие2 }
- Позволяет выполнять разные действия в зависимости от того, истинно или ложно заданное условие.
- Можно использовать дополнительные блоки else if для проверки нескольких условий.

Оператор switch-case:

- Позволяет проверять значение выражения на соответствие нескольким вариантам.
- Структура: switch (выражение) { case значение1: действие1; break; case значение2: действие2; break; default: действие_по_умолчанию; }
- Удобен для проверки множества конкретных значений.
- Может использоваться для замены многочисленных if-else.
