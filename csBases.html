Базовый синтаксис

Переменные:

- Переменная - это "контейнер", в котором вы можете хранить какое-либо значение.
Для объявления переменной указывается тип данных, за которым следует название переменной и знак равно =, после которого указывается значение, присваиваемое переменной. Каждая строка с объявлением переменной заканчивается точкой с запятой ;
- Пример:

1 int myInt = 42;

Типы данных:

- C# - язык с сильной статической типизацией, поэтому каждая переменная имеет определенный тип данных.
- Основные типы данных:
- int (целые числа): 42, -10, 0
- float и double (числа с плавающей точкой): 3.14, -2.5, 0.0
- bool (логический тип): true или false
- char (символьный тип): 'A', '5', '+'
- string (текстовый тип): "Привет, мир!"
- Пример:

1 int myInt = 32;
2 float myFloat = 3.14f;
3 bool myBool = true;
4 char myChar = 'A';
5 string myString = 'Привет!'

Ввод и вывод данных

- Для вывода данных используется Console.WriteLine().
- Для ввода данных используется Console.ReadLine().
- Пример:

1 Console.WriteLine("Введите ваше имя:");
2 string myString = Console.ReadLine();
2 Console.WriteLine("Привет, name + "!");

Арифметические и логические операторы:

- Арифметические операторы: +, -, *, /, % (остаток от деления).
- Логические операторы: <, >, <=, >=, ==, !=, && (and), || (or), ! (not).
- Пример:

1 int a = 10, b = 4;
2 int sum = a + b;
3 bool isGreater = a > b;
4 bool isEven = (b % 2 == 0);
