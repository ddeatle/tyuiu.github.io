Объектно-Ориентированное Программирование

Классы и объекты

Объявление класса

Классы - это основа объектно-ориентированного программирования. Они служат шаблоном для создания объектов, которые содержат данные (поля) и методы (функции):

 1 class Person
 2 {
 3     string name;
 4     int age;
 5     public void Greet()
 6     {
 7         Console.WriteLine($"Привет, меня зовут {name} и мне {age} лет!");
 8     }
 9 }

Создание объектов

Для создания объекта класса используется оператор new:

 1 Person person = new Person();
 2 person.name = "Иван";
 3 person.age = 30;
 4 person.Greet();

Конструкторы

Конструкторы - это методы, которые автоматически вызываются при создании объекта. Они служат для инициализации объекта:

 1 class Person
 2 {
 3     string name;
 4     int age;
 5     public Person(string n, int a)
 6     {
 7         name = n;
 8         age = a;
 9     }
10     public void Greet()
11     {
12         Console.WriteLine($"Привет, меня зовут {name} и мне {age} лет!");
13     }
14 }
15 Person person = new Person("Иван", 30);
16 person.Greet();
