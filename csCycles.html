Циклические конструкторы

Цикл do-while:

- Похож на while, но проверка условия выполняется в конце каждой итерации: do { // код внутри цикла } while (condition)
- Гарантирует, что код внутри цикла будет выполнен хотя бы один раз

Цикл while:

- Выполняет код, пока истинно указанное условие: while (condition) { // код внутри цикла }
- Используется, когда количество итераций неизвестно заранее
- Важно следить, чтобы условие в конечном итоге стало ложным, иначе цикл будет выполняться бесконечно
