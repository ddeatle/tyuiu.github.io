Работа с массивами

Объявление и инициализация массивов

Массивы в C# позволяют хранить несколько значений одного и того же типа. Массивы объявляются с помощью квадратных скобок [] после типа данных:

1 int[] numbers = new int[5];
2 string[] names = { "Иван", "Мария", "Андрей" };

Доступ к элементам массива

Элементы массива доступны через индексы, начиная с 0:

1 string[] names = { "Иван", "Мария", "Андрей" };
2 string firstPerson = names[0];
3 string secondPerson = names[1];

Изменение элементов массива

Значения элементов массива можно изменять, обращаясь к ним по индексу:

1 int[] numbers = { 1, 2, 3, 4, 5 };
2 numbers[2] = 10;

Свойства и методы массивов

Массивы в C# предоставляют различные свойства и методы, такие как Length, Sort(), Reverse():

1 int[] numbers = { 3, 1, 4, 1, 5, 9, 2, 6, 5 };
2 int length = numbers.Length;
3 Array.Sort(numbers);
4 Array.Reverse(numbers);

Работа с многомерными массивами

C# также поддерживает многомерные массивы, которые хранят данные в виде таблицы:

1 int[,] grid = new int[3,3];
2 grid[0,0] = 1;
3 grid[1,1] = 5;
4 int value = grid[2,2];
