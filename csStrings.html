Работа со строками

Объявление и инициализация строк

В C# строки представлены типом string. Строки можно инициализировать с помощью литералов в одинарных или двойных кавычках:

1 string message = "Привет, мир!";

Конкатенация строк

Строки можно объединять с помощью оператора + или с использованием интерполяции строк:

1 string firstName = "Иван";
2 string lastName = "Иванов";
3 string fullName = firstName + " " + lastName;
4 string formattedName = $"Меня зовут {firstName} {lastName}";

Доступ к символам строки

Можно обращаться к символам строки по индексам, начиная с 0:

1 string message = "Привет, мир!";
2 char firstChar = message[0];

Свойства и методы строк

Строки предоставляют множество полезных свойств и методов, таких как Length, ToUpper(), ToLower():

1 string message = "Привет, мир!";
2 int length = message.Length;
3 string uppercase = message.ToUpper();
4 string lowercase = message.ToLower();

Сравнение строк

Строки можно сравнивать с использованием операторов сравнения или метода Equals():

1 string str1 = "apple";
2 string str2 = "banana";
3 bool isEqual = str1 == str2;
