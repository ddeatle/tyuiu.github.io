Наследование, инкапсуляция, полиморфизм

Наследование

Наследование позволяет создавать новые классы на основе существующих, наследуя их свойства и методы:

 1 class Employee : Person
 2 {
 3     string jobTitle;
 4     public Employee(string n, int a, string t) : base(n, a)
 5     {
 6         jobTitle = t;
 7     }
 8     public void Introduce()
 9     {
10         Console.WriteLine($"Привет, меня зовут {name} и я работаю на должности {jobTitle}.");
11     }
12 }
13 Employee employee = new Employee("Иван", 35, "Менеджер");
14 employee.Greet();
15 employee.Introduce();

Инкапсуляция

Инкапсуляция - это механизм, который позволяет скрывать внутреннюю реализацию объекта и предоставлять доступ к нему через публичные методы:

 1 class BankAccount
 2 {
 3     private double balance;
 4     public void Deposit(double amount)
 5     {
 6         balance += amount;
 7     }
 8     public void Withdraw(double amount)
 9     {
10         balance -= amount;
11     }
12     public double GetBalance()
13     {
14         return balance;
15     }
16 }
17 BankAccount account = new BankAccount();
18 account.Deposit(1000);
19 double currentBalance = account.GetBalance();

Полиморфизм

Полиморфизм - это концепция, позволяющая объектам различных классов иметь один и тот же метод, но реализовывать его по-разному.

 1  abstract class Animal
 2 {
 3     public virtual void MakeSound()
 4     {
 5         Console.WriteLine("Животное издает звук");
 6     }
 7 }
 8 
 9  class Dog : Animal
10 {
11     public override void MakeSound()
12     {
13         Console.WriteLine("Гав-гав!");
14     }
15 }
16 
17  class Cat : Animal
18 {
19     public override void MakeSound()
20     {
21         Console.WriteLine("Мяу-мяу!");
22     }
23 }

В этом примере классы Dog и Cat являются наследниками абстрактного класса Animal. Каждый из этих классов реализует свой собственный метод MakeSound(), который выводит соответствующий звук. Таким образом, объекты разных классов могут иметь один и тот же метод, но реализовывать его по-разному, что и является проявлением полиморфизма.
